[Home](../../../index.md) >

@brettz9/no-literal-call

リテラルの関数呼び出しを禁止します。

# オプション

オプションはありません。

# 正しい例

```javascript
/* eslint @brettz9/no-literal-call: "error" */

foo();
foo`foo`;
```

# 間違った例

```javascript
/* eslint @brettz9/no-literal-call: "error" */

42();
"foo"();
true`foo`;
```
