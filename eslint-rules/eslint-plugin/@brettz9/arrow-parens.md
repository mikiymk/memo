@brettz9/arrow-parens

アロー関数式の引数リストを括弧で囲むように要求します。
ただし、引数が始め括弧の直後にある場合を除きます。

# オプション

オプションはありません。

# 正しい例

<!-- prettier-ignore -->
```javascript
/* eslint @brettz9/arrow-parens: "error" */

var foo = (x) => x;
var foo = (x => x);
array.reduce(x => x, 0);
```

# 間違った例

<!-- prettier-ignore -->
```javascript
/* eslint @brettz9/arrow-parens: "error" */

var foo = x => x;
var foo = ((x) => x);
var foo = (
  x => x
);
array.replace(/./g, x => x);
```
