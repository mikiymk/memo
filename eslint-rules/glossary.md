# `var` 宣言

```javascript
var foo;
var foo = 42;
var foo = 42, bar;
```

# `let` 宣言

```javascript
let foo;
let foo = 42;
let foo = 42, bar;
```

# `const` 宣言

```javascript
const foo = 42;
const foo = 42, bar = 84;
```

# 関数宣言

```javascript
function foo(arg) {
  return arg;
}
```

# `if` 文

```javascript
if (isTrue) {
  do1();
} else if (isFalse) {
  do2();
} else {
  do3();
}
```
