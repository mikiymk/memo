# `var` 宣言

```javascript
var foo;
var bar = 42;
var baz = 42, bam;
```

# `let` 宣言

```javascript
let foo;
let bar = 42;
let baz = 42, bam;
```

# `const` 宣言

```javascript
const bar = 42;
const baz = 42, bam = 84;
```

# 関数宣言

```javascript
function foo(arg) {
  return arg;
}
```

# `if` 文

```javascript
if (isTrue) {
  do1();
} else if (isFalse) {
  do2();
} else {
  do3();
}
```
